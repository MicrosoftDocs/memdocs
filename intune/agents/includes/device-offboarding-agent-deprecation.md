> [!NOTE]
> **The Device Offboarding Agent will no longer be available starting May 31, 2026.** We recommend reviewing existing offboarding processes and transitioning to alternative supported device lifecycle and remediation options in Intune ahead of May 31, 2026. Please see the details below:
>
>**Device Offboarding Agent timeline**
>
>- **April 30, 2026**: The Device Offboarding Agent can no longer be set up.
>- **May 31, 2026**: The Device Offboarding Agent is removed from the Intune admin center and is no longer available.
>
>**What this means for customers**
>
>- Customers who have already set up the Device Offboarding Agent can continue using it **until May 31, 2026**.
>- If the agent is deleted between April 30 and May 31, 2026, it cannot be set up again.
>- As of **May 31, 2026**, the Device Offboarding Agent will no longer be accessible.
>
>**Recommended actions**
>
>- Complete any active offboarding actions before **May 31, 2026**.
>- Avoid building new dependencies on the Device Offboarding Agent.
>- Review existing offboarding processes and transition to supported device lifecycle and remediation options in Intune ahead of **May 31, 2026**.

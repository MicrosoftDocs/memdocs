## Enable the agent

To enable the Device Offboarding Agent, follow these steps:

1. In the [Microsoft Intune admin center][INT-AC], select **Agents** and select the agent you want to enable.
1. Select **Set up Agent** to open the set-up pane.
1. Review the details to ensure requirements are in place, then select **Start agent**.

The agent runs until it finishes and then displays its results in the **Overview** tab.

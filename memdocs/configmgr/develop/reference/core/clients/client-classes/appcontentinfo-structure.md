# AppContentInfo Structure
In Configuration Manager, the `AppContentInfo` structure contains information about the application content.  

## Syntax  

```  
struct AppContentInfo  
{  
    LPCWSTR szContentId;  
    LPCWSTR szContentVersion;  
    LPCWSTR szLocalPath;  
};  
```  

## Members  
 `szContentId`  
 The content id.  

 `szContentVersion`  
 The content version.  

 `szLocalPath`  
 The local path.
